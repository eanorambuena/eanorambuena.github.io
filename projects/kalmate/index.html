Kalmate: Terminal Financiera Kalmate: Financial Terminal

Terminal financiera estilo Bloomberg con datos de mercado en tiempo real vía Yahoo Finance API. Construido con Nuxt 3, Vue 3, TypeScript y TradingView charts. Dashboard categorizado (Equities, Forex, Bonds, Commodities, Crypto), portafolio con P&L, alertas inteligentes con notificaciones del navegador, screener con filtros y presets, noticias financieras, y pipeline visual de nodos para análisis de datos (Filtro de Kalman, SMA, RSI, EMA, Forecast). Cacheo inteligente del servidor y atajos de teclado para navegación eficiente. Sin necesidad de API keys. Bloomberg-style financial terminal with real-time market data via Yahoo Finance API.
