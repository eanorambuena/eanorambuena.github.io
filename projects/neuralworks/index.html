NeuralWorks — Fullstack POS NeuralWorks — Fullstack POS

Trabajé como Fullstack Software Engineer en el proyecto HOP de NeuralWorks, un Point of Sale (POS) para Tablets. Lideré la migración de la UI legacy de React Native Paper a Tamagui, diseñé e implementé el módulo completo de inventario (cliente + servidor), y optimicé la infraestructura cloud con AWS, SST y Docker. También implementé Cursor Rules que redujeron los costos de IA de >$200/mes a $0, y establecí flujos de Atomic PRs para eliminar cuellos de botella en el equipo.
