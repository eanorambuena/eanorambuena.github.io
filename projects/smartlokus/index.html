Smartlokus: GIS para Minería Smartlokus: GIS App for Mining

Aplicación GIS de visualización de datos geoespaciales construida para Acofuz, utilizada por Procel y Antofagasta Minerals. Frontend desarrollado desde cero con React, TypeScript y Vite, con arquitectura escalable y mantenible. Integración vía API con backend Python para procesamiento de datos geoespaciales. GIS visualization app built for Acofuz, used by Procel and Antofagasta Minerals. Frontend built from scratch with React, TypeScript, and Vite, with scalable and maintainable architecture. API integration with Python backend for geospatial data processing.
